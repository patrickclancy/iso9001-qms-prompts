# Decision Tree

## I have a Word document and need to analyze it

Use Prompt 00.

## I have a before document and matching after document

Use Prompt 00 on both documents first. Then use Prompt 01.

## I have a comparison and need reusable rules

Use Prompt 02.

## I have rules from all five document pairs

Use Prompt 03.

## I have an approved rulebook and need a reusable reviewer prompt

Use Prompt 04.

## I have a new document and want to know what is wrong with it

Use Prompt 00 first. Then use Prompt 05.

## I have review findings and want to rewrite the document

Use Prompt 06.

## I have a revised Word document and want to check formatting

Use Prompt 00 on the revised document. Then use Prompt 07.

## I want a final release-readiness check

Use Prompt 08.

## The AI says the change may affect meaning

Stop and ask the process owner, Quality, or Document Control.

## The AI says the formatting requirement is unclear

Stop and ask Document Control to define the requirement.
