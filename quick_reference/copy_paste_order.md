# Copy/Paste Order

Use this page when you only need to know which prompt comes next.

## Learning phase

```text
1. Copy prompts/00_document_audit_packet/prompt.md
   Use it on each before and after Word document.

2. Copy prompts/01_pairwise_transformation_analysis/prompt.md
   Use it on each before/after audit packet pair.

3. Copy prompts/02_rule_extraction/prompt.md
   Use it on each transformation analysis.

4. Copy prompts/03_cross_pair_rule_consolidation/prompt.md
   Use it on all five extracted rule files.

5. Human approval of the master rulebook.

6. Copy prompts/04_reviewer_rulebook_generation/prompt.md
   Use it on the approved master rulebook.
```

## Production phase

```text
1. Copy prompts/00_document_audit_packet/prompt.md
   Use it on the new before Word document.

2. Copy prompts/05_production_review/prompt.md
   Use it to review the new document.

3. Copy prompts/06_rewrite_before_to_after/prompt.md
   Use it to rewrite the document.

4. Create or update the Word document.

5. Copy prompts/00_document_audit_packet/prompt.md
   Use it on the revised Word document.

6. Copy prompts/07_word_formatting_compliance/prompt.md
   Use it to check formatting.

7. Copy prompts/08_final_qa_gate/prompt.md
   Use it for final QA.
```
