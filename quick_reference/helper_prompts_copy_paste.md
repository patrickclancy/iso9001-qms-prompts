# Helper Prompts Quick Copy/Paste Guide

Use this when you only need a small edit.

## Basic copy/paste order

```text
1. Choose the helper from helper_prompts/HELPER_SELECTION_GUIDE.md.
2. Open the helper folder.
3. Copy prompt.md.
4. Paste the prompt into your AI tool.
5. Paste the short section, paragraph, step, or table row below the prompt.
6. Review the result.
7. Use the meaning guard helper if the text touches requirements, records, responsibilities, approvals, or scope.
8. Paste only approved text into Word.
```

| Situation | Helper |

| Clean up any short paragraph | `00_general_section_polisher` |

| Scope section | `02_scope_section_helper` |

## Fast input block

```text
Document type:

Approved rulebook or style rules to apply:
Company terminology that must be preserved:

Specific concern:
```

## When to stop and ask for approval

Ask the process owner or Quality reviewer before accepting a change that affects:

- Required actions.
- Responsibilities.
- Approvals.
- Records or evidence.
- Retention.
- Scope.
- Regulatory, statutory, customer, or QMS commitments.
- Process sequence.
