# Common Input Template for Helper Prompts

Copy this block below any helper prompt.

```text
Document type:
[Policy / Procedure / Work Instruction / Form / Other]

Document title:
[Paste the document title]

Section being edited:
[Purpose / Scope / Key Requirements / Responsibilities / Process Step / Records / Other]

Current text to improve:
[Paste the exact text. Use a short section, paragraph, table row, or step.]

Approved rulebook or style rules to apply:
[Paste relevant rule IDs or style rules. If you do not have a rulebook yet, write "Not available yet — use clear, formal QMS tone and flag assumptions."]

Company terminology that must be preserved:
[Paste role names, department names, product names, software names, acronyms, or write "None"]

Words or meaning that must not change:
[Paste required obligations, responsibilities, records, approval requirements, scope limits, or write "None known"]

Specific concern:
[Grammar / tone / too long / unclear responsibility / unclear sequence / too informal / formatting-ready wording / other]

Output preference:
[Minimal edit / moderate polish / give me two options]
```

## Tips

- Paste less text, not more.
- Do one section at a time.
- Keep the original text so you can compare it to the revised text.
- Use the meaning guard helper when the revised wording affects requirements, responsibilities, approvals, records, or process sequence.
