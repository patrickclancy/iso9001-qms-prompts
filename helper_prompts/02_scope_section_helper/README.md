# README — Scope Section Helper

## What this helper does

Use this to make a Scope section clearer by defining applicable roles, departments, sites, processes, boundaries, inclusions, and exclusions.

## Do not use it for

Do not use this to add new scope, remove scope, or guess exclusions.

## What to prepare

Prepare the current Scope text, known boundaries, applicable departments or roles, and any explicit exclusions.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that the revised scope did not accidentally include or exclude people, sites, products, activities, or records.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
