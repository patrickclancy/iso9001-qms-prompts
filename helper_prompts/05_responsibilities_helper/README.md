# README — Responsibilities Helper

## What this helper does

Use this to clarify who is responsible for what in a policy, procedure, work instruction, or table.

## Do not use it for

Do not use this to assign new responsibilities or change approval authority unless that change is explicitly approved.

## What to prepare

Prepare the current responsibility text and a list of approved role names.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that no role, owner, reviewer, approver, or verifier changed unintentionally.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
