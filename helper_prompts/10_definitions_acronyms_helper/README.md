# README — Definitions and Acronyms Helper

## What this helper does

Use this to clean up definitions, terms, acronyms, and abbreviation lists.

## Do not use it for

Do not use this to redefine controlled terms, regulatory terms, or customer-specific terms without approval.

## What to prepare

Prepare the current definitions, acronyms, approved terminology, and any terms that must not change.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that definitions are concise and that no technical meaning changed.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
