# README — Table Content Polisher

## What this helper does

Use this to polish wording inside responsibility tables, records tables, requirements tables, or small data tables.

## Do not use it for

Do not use this for Word table styling, borders, shading, or layout. Use the main formatting compliance prompt for that.

## What to prepare

Prepare the table row, small table, column names, and any terms that must be preserved.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that no row, record, role, approval, or requirement changed unintentionally.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
