# README — Process Step Polisher

## What this helper does

Use this to clean up one procedure step or a small group of closely related steps.

## Do not use it for

Do not use this to redesign the process or create a full procedure.

## What to prepare

Prepare the current step, nearby step numbers if needed, approved role names, and any required records.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that sequence, responsibility, timing, approvals, and evidence requirements did not change.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
