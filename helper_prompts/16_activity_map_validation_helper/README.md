# README — Activity Map Validation Helper

## What this helper does

Use this after creating a process activity map to confirm that it matches the written procedure steps.

## Do not use it for

Do not use this to create a new process map from scratch. Use the Process Activity Map Helper first.

## What to prepare

Prepare the written steps and the draft activity map.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check all mismatches before placing the map into the controlled document.

## Recommended next step

Route major mismatches to the process owner before updating the document.
