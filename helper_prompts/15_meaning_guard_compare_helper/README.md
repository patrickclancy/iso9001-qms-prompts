# README — Meaning Guard Compare Helper

## What this helper does

Use this after any helper when you need to confirm that the polished wording did not change QMS meaning.

## Do not use it for

Do not use this as the first editing prompt. Use it after you have original and revised text.

## What to prepare

Prepare the original text, revised text, and any terms, obligations, records, roles, or approvals that must not change.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check the decision and do not use the revised text if the helper says it needs review or should not be used.

## Recommended next step

Use the safer revised wording if provided, then route any flagged item to the appropriate reviewer.
