# README — Process Activity Map Helper

## What this helper does

Use this to create a simple process activity map, box-and-arrow flow, or swimlane table from existing process steps.

## Do not use it for

Do not use this to design a new process or add steps that are not in the source text.

## What to prepare

Prepare the procedure steps, role names, decisions, inputs, outputs, records, and known start/end points.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that the map includes no invented activities and that decisions, handoffs, and records match the written procedure.

## Recommended next step

After creating the map, use `16_activity_map_validation_helper` to verify that the map matches the procedure steps.
