# README — Inputs, Outputs, and Records Helper

## What this helper does

Use this to clean up sections that describe inputs, outputs, records, objective evidence, or retained documented information.

## Do not use it for

Do not use this to create new records or retention requirements unless they are provided by the process owner or Quality.

## What to prepare

Prepare record names, systems, form names, owners, retention details, and the current section text.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that no record was removed and no retention requirement changed unintentionally.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
