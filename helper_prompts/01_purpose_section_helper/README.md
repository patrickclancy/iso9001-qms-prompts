# README — Purpose Section Helper

## What this helper does

Use this to clean up or draft a short Purpose section for a policy, procedure, work instruction, or controlled QMS document.

## Do not use it for

Do not use this to define scope boundaries, process steps, responsibilities, or records.

## What to prepare

Prepare the current Purpose text, document title, document type, and the process or outcome the document supports.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that the Purpose is short, clear, accurate, and does not add unsupported requirements or commitments.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
