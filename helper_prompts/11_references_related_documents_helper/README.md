# README — References and Related Documents Helper

## What this helper does

Use this to clean up references, related documents, forms, templates, and associated document lists.

## Do not use it for

Do not use this to add standards, clauses, or related documents that are not provided.

## What to prepare

Prepare the current reference list, document IDs, titles, and any approved naming format.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that no valid reference was removed and no unsupported reference was added.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
