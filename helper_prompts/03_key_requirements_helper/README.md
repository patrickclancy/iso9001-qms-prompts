# README — Key Requirements Helper

## What this helper does

Use this to polish requirement bullets, shall/must statements, audit-ready requirements, or compliance expectations.

## Do not use it for

Do not use this to create new requirements without a source.

## What to prepare

Prepare the current requirements, any known responsible roles, timing, records, and approved obligation-word rules.

## How to use

1. Open `prompt.md`.
2. Copy the full prompt.
3. Paste it into your AI tool.
4. Fill in the input block at the bottom.
5. Review the proposed wording.
6. Copy only the approved wording into the Word document.

## What to check before accepting the output

Check that shall/must/should/may meanings are unchanged and that no required record, review, or approval was removed.

## Recommended next step

Use the meaning guard helper if the proposed wording changes obligations, responsibilities, approvals, records, scope, or process sequence.
