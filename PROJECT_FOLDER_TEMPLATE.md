# Recommended Project Folder Template

Create a project folder like this on your computer.

```text
QMS_Document_Reviewer_Project/
├── 00_Source_Documents/
│   ├── Pair01_Before.docx
│   ├── Pair01_After.docx
│   ├── Pair02_Before.docx
│   ├── Pair02_After.docx
│   ├── Pair03_Before.docx
│   ├── Pair03_After.docx
│   ├── Pair04_Before.docx
│   ├── Pair04_After.docx
│   ├── Pair05_Before.docx
│   └── Pair05_After.docx
├── 01_Audit_Packets/
├── 02_Transformation_Analysis/
├── 03_Extracted_Rules/
├── 04_Master_Rulebook/
├── 05_Production_Reviewer_Prompt/
├── 06_New_Document_Reviews/
├── 07_Rewritten_Documents/
├── 08_Formatting_Checks/
└── 09_Final_QA/
```

## Where each output goes

| Output type | Folder |
|---|---|
| Audit packets | 01_Audit_Packets |
| Pairwise transformation analysis | 02_Transformation_Analysis |
| Extracted rules | 03_Extracted_Rules |
| Master rulebook | 04_Master_Rulebook |
| Production reviewer prompt | 05_Production_Reviewer_Prompt |
| New document review findings | 06_New_Document_Reviews |
| Rewritten document content | 07_Rewritten_Documents |
| Formatting check reports | 08_Formatting_Checks |
| Final QA reports | 09_Final_QA |
